# Skylight Calendar Card Wiki

Welcome to the Skylight Calendar Card documentation.

## What this card does

- Displays one or more Home Assistant calendars in a polished family-calendar layout
- Supports month and week-style views
- Includes optional event create/edit/delete controls (integration support varies)
- Provides flexible styling and layout controls
